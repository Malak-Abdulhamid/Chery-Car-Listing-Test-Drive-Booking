# Chery Car Listing Test Drive Booking 

A modern and responsive car showcase website for **Chery** vehicles.
The website presents different Chery car models with attractive UI design, vehicle images, and downloadable PDF brochures for each car.

# Project Overview

This project is a frontend website designed to display Chery car models in an organized and user-friendly way.
Users can browse available vehicles, explore car details, and download brochures directly from the website.

The website focuses on:

* Clean and modern design
* Responsive layout
* Easy navigation
* Interactive user experience
* Car model presentation


# Features

*  Display multiple Chery car models
*  Download car brochures in PDF format
*  High-quality car images
*  Responsive design for mobile and desktop
*  Interactive elements using JavaScript
*  Modern UI using CSS styling

1. Download or clone the project.
2. Open the project folder.
3. Run the `index.html` file in your browser.
